CSV Format Requirements

Your CSV file must include the following exact column headers (case-sensitive):

  • Type – e.g., "ITR"
  • Posting Date – in YYYY-MM-DD format
  • Document Number
  • Item No.
  • Ordered Quantity
  • Delivered Quantity
  • Remaining Open Qty
  • From Warehouse Code
  • To Warehouse
  • Remarks (optional)
  • Special Instruction (optional)
  • Branch Name
  • Branch Name v2 (optional)
  • Document Status – e.g., "C" or "O"
  • Due Date – in YYYY-MM-DD format
  • User_Code
  • PO Number (optional)
  • ISMS SO#
  • CBM
  • Customer/Vendor Code (optional)
  • Customer/Vendor Name (optional)

⚠️ Important: Use the downloadable template to avoid formatting errors.

✅ Duplicate Prevention: Records are validated against existing data using the combination of Document Number, Item No., and Ordered Quantity. Duplicate rows will be automatically skipped during upload.
